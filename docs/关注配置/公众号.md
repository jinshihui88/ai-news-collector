# 如何配置关注的公众号
配置文件在 [config/wechat-accounts.json](/config/wechat-accounts.json)

**配置格式**:
```json
[
  {
    "fakeid": "MzU0MTkwOTUyMA==",
    "nickname": "骁哥AI编程"
  },
  {
    "fakeid": "********",
    "nickname": "其他公众号"
  }
]
```

**字段说明**:

| 字段 | 必需 | 说明 |
|------|------|------|
| `fakeid` | ✅ 是 | 公众号的唯一标识符 |
| `nickname` | ✅ 是 | 公众号名称 |
| `description` | ❌ 否 | 备注说明 |

**如何获取 Fakeid**:

1. **登录公众号后台**: https://mp.weixin.qq.com/

登入公众号后台

![](https://cdn.ziliu.online/images/2025/11/03835182-21f2-4fb2-8d31-2725d8c1ba0f.jpg)

2. **新建文章**:

新建一个空白文章，然后点击上面的“插入超链接”

![](https://cdn.ziliu.online/images/2025/11/d8bcdf8c-abc2-4fb5-b4d8-c217858f10c7.jpg)

3. **点击选择其他账号**:

点击选择其他账号

![](https://cdn.ziliu.online/images/2025/11/4f18285d-082a-40f8-971b-ceaf572e9823.jpg)


4. **获取 Fakeid**:
   - 按 `F12` 打开浏览器开发者工具
   - 切换到 Network 标签
   - 过滤 `searchbiz` 请求
   - 在搜索结果中查找请求
   - 复制 fakeid 值
  
![](https://cdn.ziliu.online/images/2025/11/b8d640c5-ac81-4edd-9e15-b349a29747fa.jpg)


5. **配置 [config/wechat-accounts.json](/config/wechat-accounts.json)**
在关注账号列表中，添加新的公众号，比如
```json
[
  {
    "fakeid": "MzU0MTkwOTUyMA==",
    "nickname": "骁哥AI编程"
  },
  {
    "fakeid": "********",
    "nickname": "其他公众号A"
  },
    {
    "fakeid": "********",
    "nickname": "其他公众号B"
  }
]
```
