# 如何配置关注的星球号
配置文件在 [config/zsxq-groups.json](/config/zsxq-groups.json)

**配置格式**:
```json
[
  {
    "groupId": "15552****5212",
    "groupName": "你关注的星球号",
    "hashtags": [
      { "id": "15555****55522", "name": "AI风向标" },
      { "id": "15555****55523", "name": "其他话题" }
    ]
  }
]
```

**字段说明**:

| 字段 | 必需 | 说明 |
|------|------|------|
| `groupId` | ✅ 是 | 星球的唯一标识符(从星球详情页 URL 获取) |
| `groupName` | ✅ 是 | 星球名称 |
| `hashtags` | ✅ 是 | 需要采集的话题数组,每项填写 `{ id, name }` |

**如何配置**:

**步骤一：登录知识星球**

登录你的知识星球 [https://wx.zsxq.com/](https://wx.zsxq.com/)

![](https://cdn.ziliu.online/images/2025/11/6aaf6b3d-7558-4290-ae20-4fa12fe3aade.jpg)

**步骤二：切到你关注的星球，并通过链接获取groupId**

切到你关注的星球，并通过链接获取groupId

![](https://cdn.ziliu.online/images/2025/11/19d8fd12-b175-4ee2-b01b-34b07e0bd044.jpg)

**步骤三：切到你关注的话题，并获取hashtags**
   - 按 `F12` 打开浏览器开发者工具
   - 切换到 Network 标签
   - 过滤 `hash` 请求
   - 在搜索结果中查找请求
   - 复制 hashtags 值

![](https://cdn.ziliu.online/images/2025/11/8a84f679-d17e-4829-920e-08eaa0a532f2.jpg)

**步骤四：在[config/zsxq-groups.json](/config/zsxq-groups.json)中添加**
添加至关注配置列表，举例：
```json
[
  {
    "groupId": "15552****5212",
    "groupName": "你关注的星球1",
    "hashtags": [
      { "id": "15555****55522", "name": "A话题" },
      { "id": "15555****55523", "name": "B话题" }
    ]
  },
    {
    "groupId": "15552****5212",
    "groupName": "你关注的星球2",
    "hashtags": [
      { "id": "15555****55522", "name": "C话题" },
    ]
  }
]
```

--- 
**⚠️注意事项** 必须配置你已加入的星球
